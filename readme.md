# 数据结构与算法学习笔记
## 时间复杂度
### 概念
时间复杂度代表的是随着数据量增长，代码执行时间的变化趋势
### 分析方法

#### 只关注循环执行次数最多的一段代码
我们在分析一个算法、一段代码的时间复杂度的时候，也只关注循环执行次数最多的那一段代码就可以了
####  加法法则：总复杂度等于量级最大的那段代码的复杂度
O(1)+O(n)+O(n**2)=O(n**2)
int cal(int n) {
   int sum_1 = 0;
   int p = 1;
   for (; p < 100; ++p) {         //    O(1)
     sum_1 = sum_1 + p;
   }

   int sum_2 = 0;
   int q = 1;
   for (; q < n; ++q) {      //O(n)
     sum_2 = sum_2 + q;
   }
 
   int sum_3 = 0;
   int i = 1;
   int j = 1;
   for (; i <= n; ++i) {  // O(n**2)
     j = 1;  
     for (; j <= n; ++j) {
       sum_3 = sum_3 +  i * j;
     }
   }
 
   return sum_1 + sum_2 + sum_3;
 }
